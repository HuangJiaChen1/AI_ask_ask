已知脆弱点（文档标注）

1. continue_conversation 庞大嵌套 (~1000+ 行)
6 条互斥路径，内联 async def 生成器，大量重复 StreamChunk 构建代码。
2. 桥接状态机分散在三处
HTTP 层、PaixuejiAssistant、graph.py 开场节点各管一段。
3. 意图分类重复与不一致
图分析节点做大写分类，属性管道在 HTTP 层独立做小写分类，action_subtype 只在图中使用。
4. 全局事件循环线程安全
同步 Flask 路由中跑异步代码，async_gen_to_sync 转换，硬编码 10 秒超时。
5. 趣闻功能悬浮
node_generate_fun_fact 已定义但未接入主工作流（build_paixueji_graph 未 add_node）。
